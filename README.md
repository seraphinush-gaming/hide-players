# hide-players
tera-proxy module to hide/show players

## Dependency
- `Command` module

## Usage
### `hide` · `ㅗㅑㅇㄷ`
- Toggle on/off
- Default is on
### Parameters
#### `a` · ` ㅁ`
- Toggle guild/party member on/off
- Default is on
#### `r` · `ㄱ` · `refresh`
- Refresh filter

## Info
- Original author : [wuaw](https://github.com/wuaw)
- Hides all players except guild and party/raid members
